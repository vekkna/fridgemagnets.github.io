Fridge Magnets

This site turns blocks of code into shuffled and draggable "fridge magnets" which teachers can share with their students.
This is a good way for students to focus on the logic of a program without having to worry about syntax.

Teachers

Enter your code below. Use empty lines to form separate magnets. White space will be trimmed. Decide yourself what to do with any braces.


Students

Press the button with an empty field to see how the placeholder code would look.
